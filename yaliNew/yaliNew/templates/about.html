The project is part of the "data science" profession of the 10th grade students of the Alterman High School in Tel Aviv.

The project combines two main areas of knowledge:

- Programming in the web environment.

- Data science.

As part of the project, I located a database of information on the network and intend to build a website around them that allows access to the database, data analysis and presentation to users in a convenient way. The site will allow the user to explore the data in the chosen field and try to reach valuable insights and conclusions.

The project will be based on the following technological components:

- Python language.

- Pandas Library.

- Flask infrastructure.

- MatPlotLib library.

- Requests directory.
